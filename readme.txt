== Description ==

This plugin has been developed to facilitate the visualization of the [SynField](https://www.synfield.gr/) node latest measurements in web applications based on CMS Wordpress.

== Installation ==


== Usage ==

Two modes are supported. Support 1 global API key and 1 API key per widget. The API key is per user.

Visualize data from nodes that belong in the same user
---
The admin registers the global API key (generated from us) in the plugin settings page and then, register only the node S/N in the widget admin form.

Visualize data from nodes that belong in different users
---
The admin skips the global API key in the plugin settings page. Admin register directly the user's API key and the node S/N in the widget admin form.
